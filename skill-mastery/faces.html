What is a Face?

Temple Guard 'faces' are stationary commands that are accomplished without marching movements. These movements direct Temple Guards to the command of the speaker. Be careful to only obey the host if they use the proper format: 'Right face.' Unless otherwise directed, any command that does not follow that format is considered invalid.

  • Right face. - Turn 90° to the right.
  • Left face. - Turn 90° to the left.
  • Right incline. - Turn 45° to the right.
  • Left incline. - Turn 45° to the left.
  • About face. - Turn around, making a 180° turn.
  • Center/Centre face. - Face the host's direction at the nearest 90° angle.
  • Control face. - Follow the host’s movements. This command is broken by 'Center face.' and 'Lock.'
  • Lock. - Do not follow upcoming commands. The command is broken by 'Center face.' and 'Unlock.'
  • Unlock. - Disables the 'Lock.' command. Continue following given commands. If 'Control face.' is active when the 'Lock.' command was given, continue to track the host after unlocking.

Chains

Chains are a set of subsequent commands given by a host. In order for a chain to be considered valid, all commands within the chain have to be properly formatted. If one command is incorrectly formatted, the entire chain is invalid. It is also important to look for faces such as 'center face' and 'control face' as they cancel out the previous commands, allowing the Temple Guard to start from that position. Below are examples of valid and invalid chains, some of which include special cases:

  • Right face, about face, left face, about face, right incline, about face. - This chain is VALID. When a Temple Guard finishes following the sequence, they should have turned to the right 225° from what would be considered 'Center face.'
  • Left face, right face, about face, right face. left face. - This chain is INVALID. There is a period instead of a comma. A Temple Guard should not move when this chain command is given.
  • Left face, right face, about face, center face, right incline. - This chain is VALID and contains a special circumstance. The commands: 'left face, right face, about face' do not need to be carried out by the Temple Guard because the 'center face' command follows them. When the sequence is complete, the Temple Guard will have turned to the right 45° from what would be considered "Center face."
  • Left face, right face, about face, left incline, right face, about face, right incline, left face. - This chain is VALID. When a Temple Guard finishes the sequence, they should have turned 0° from what would be considered "Center face."
  • Left face, right face, Left incline, right face, about face, right incline, left face. - This chain is INVALID. The command, "Left incline" should not be capitalized. The Temple Guard should not move when this chain command is given.
